**项目说明** 
- cloud框架结构项目(base)
<br> 
<br>

**项目结构** 
```
parent

├─ctk-common 公共实体类 
│
├─ctk-service-user
│  └─用户
│ 
├─ctk-80 
│  └─对外提供服务
│ 

```
<br> 

**技术选型：** 
- 核心框架：Spring Boot 2.0
- 安全框架：Apache Shiro 1.4
- 视图框架：Spring MVC 5.0
- 持久层框架：MyBatis 3.3
- 定时器：Quartz 2.3
- 数据库连接池：Druid 1.0
- 日志管理：SLF4J 1.7、Log4j
- 页面交互：Vue2.x
